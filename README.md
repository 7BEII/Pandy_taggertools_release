## 📖 简介

Pandy AI 打标器是一款基于 Web 的专业打标工具，利用先进的 AI 视觉模型自动生成高质量的图像描述。特别适用于：

- 🎨 **LoRA 训练**：为 Stable Diffusion LoRA 模型准备高质量训练数据
- 📊 **数据集标注**：批量处理图像并生成准确的描述标签
- 🔄 **图像编辑训练**：成对处理原图→目标图，生成编辑指令
- 💬 **AI 智能对话**：与 AI 进行图像相关的智能交互

## ✨ 功能特性

### 🏷️ 文生图反推模式

**适用场景**：通用的文生图数据集打标

**核心功能**：
- 单图上传、批量上传、文件夹递归导入
- AI 智能反推图像描述
- 实时编辑和优化提示词
- 批量重命名、裁切、格式转换
- 支持 TXT、JSON、CSV 导出

---

### 🎨 图像编辑模型反推模式

**适用场景**：图像编辑模型训练（原图→目标图成对处理）

**导入模式**：
- **默认模式**：按顺序自动配对（1.png + 2.png）
- **规则匹配模式**：按后缀匹配（001_R.png + 001_T.png）

**核心功能**：
- 自定义原图/目标图后缀（默认 _R / _T）
- AI 自动对比生成编辑指令
- 批量裁切、重命名、格式转换
- 保持配对关系导出

---

### 📊 训练分析器模式

**核心功能**：
- 支持 SD WebUI、Kohya SS 训练日志
- 自动解析损失曲线、学习率变化
- 数据集质量分析（分辨率、标签统计）
- 导出 Excel 报告

---

### 💬 AI 对话模式

**核心功能**：
- 上传图像与 AI 智能对话
- 多轮对话，保持上下文
- 自动保存对话历史
- 支持图像分析、风格讨论、标注优化

## ⚙️ 配置说明

### 配置文件

```
apikey_config/
├── apikey.json              # API 配置（敏感信息）
├── config.json              # 提示词模板配置
├── 默认单图反推.json         # 默认单图模板
└── 默认编辑模型.json         # 默认编辑模板
```

### 配置管理

- **备份配置**：定期备份 `apikey.json` 和 `config.json`
- **分享模板**：只分享 `config.json`，不要分享 `apikey.json`
- **重置配置**：删除配置文件，程序会自动创建默认配置

## ❓ 常见问题

### Q1: 程序无法启动？

**解决方案**：
1. 确认 Windows 10/11 64位系统
2. 检查杀毒软件是否拦截
3. 以管理员身份运行
4. 查看控制台错误信息

### Q2: 浏览器没有自动打开？

**解决方案**：
1. 手动打开浏览器访问 `http://localhost:5000`
2. 检查防火墙设置

### Q3: API 调用失败？

**解决方案**：
1. 检查 API Key 是否正确
2. 确认网络连接正常
3. 验证 API 提供商服务状态
4. 检查 API 额度是否充足

### Q4: 图像无法上传？

**解决方案**：
1. 确认图像格式（支持 JPG、PNG、WebP）
2. 检查图像大小（建议小于 10MB）
3. 确认文件路径没有特殊字符
4. 尝试重新启动程序

### Q5: 如何获取激活码？

**获取方式**：
- 💬 联系微信：**PQB0660**
- 📖 详细文档：[使用文档](https://ai.feishu.cn/wiki/WI3HwYpBAiJ4ZPkeB0ScWxiTn2f?from=from_copylink)
- 提供机器码获取激活码
- 激活码格式：20 位字符（示例：`ECC02-B268B-D868E-FPDYY`）

### Q6: 如何获取 API Key？

**获取方式**：
- **SiliconFlow**：https://siliconflow.cn
- **ModelScope**：https://modelscope.cn
- **其他提供商**：查看各自官网

### Q7: 图像编辑模式如何正确配对？

**解决方案**：
1. 使用规则匹配模式
2. 按照命名规范：`001_R.png` + `001_T.png`
3. 确保原图和目标图的前缀一致
4. 检查配对结果中的绿色标记

### Q8: 如何更新软件？

**更新步骤**：
1. 备份 `apikey_config/` 目录
2. 下载新版本
3. 替换程序文件
4. 恢复配置文件
5. 激活码无需重新获取
