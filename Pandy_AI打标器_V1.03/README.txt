# Pandy AI Tagger

## Quick Start / 快速开始

### Launch Application / 启动应用
Double-click `Start_Pandy_Tagger.bat` to start the application.
双击 `Start_Pandy_Tagger.bat` 即可启动应用。

The browser will open automatically after startup.
应用启动后会自动打开浏览器访问本地服务。

### First Time Setup / 首次使用
1. Click the "Settings" button in the top right corner
   点击右上角的"设置"按钮
   
2. Configure your API Key in the "API Configuration" tab
   在"API配置"标签页中配置您的 API Key
   
3. Supported providers / 支持的服务商：
   - SiliconFlow
   - ModelScope
   - Tuzi API

### Features / 功能说明
- **Single Image Tagging**: Use AI to analyze images and generate description tags
  **单图反推**: 使用 AI 模型分析图片并生成描述标签
  
- **Edit Model**: Compare two images and generate transformation prompts
  **编辑模型**: 对比两张图片，生成转换提示词
  
- **Training Analysis**: Analyze training logs and generate visual reports
  **训练分析**: 分析训练日志，生成可视化报告
  
- **Batch Processing**: Support batch import and processing of images
  **批量处理**: 支持批量导入和处理图片

### Directory Structure / 目录说明
- `apikey_config/`: API configuration files / API 配置文件目录
- `training_datas/`: Training data directory / 训练数据目录
- `training_edit_tmp/`: Edit mode temporary files / 编辑模式临时文件
- `ai_chat/`: AI conversation history / AI 对话历史记录

### Notes / 注意事项
- API Key configuration is required for first use
  首次使用需要配置 API Key
  
- Ensure network connection is available
  确保网络连接正常
  
- Regular backup of configuration files is recommended
  建议定期备份配置文件
